隱私政策

最後更新：2026年5月11日

1. 資料蒐集

本應用程式（小赫廣告指揮中心）為內部廣告管理工具，僅供授權帳號使用，不蒐集一般用戶個人資料。

2. Meta 資料存取

本應用程式透過 Meta Marketing API 存取廣告帳號資料，包含廣告活動成效、Instagram 洞察報告。所有資料存取均需用戶授權，且僅用於廣告管理目的。

3. 資料使用

存取的資料僅用於：顯示廣告成效報告、建立廣告草稿。不與第三方共享，不用於商業分析以外之目的。

4. 資料保存

本應用程式不永久儲存從 Meta API 取得的資料，所有資料均即時查詢呈現。
